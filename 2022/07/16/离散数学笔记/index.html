命题与联结词

1、命题的概念

  • 命题是判断真假的陈述句

  • 命题符号化：

    • 将自然语言转化为命题公式

Ex:

  • True: 2 + 3 = 5;
  • False: 2 - 3 = 5;

2、命题分类

(1) 原子命题

  • 命题标识符：表示原子命题的符号

    • 命题常元
    • 命题变元

(2) 复合命题

  • 经联结词联结而成

  • 由两个或两个以上原子命题组成的命题

  • Ex：如果温度在0℃以下，则水会结冰

3、命题联结词

(1) 否定联结词

  • 逻辑非

(2) 合取联结词

  • 逻辑与
  • “同时取真”关系

(3) 析取联结词

  • 逻辑或
  • “选择取真”关系
  • 析取词用于“可兼或”

(4) 蕴涵联结词

  • “除非前真后假”关系：

    • 善意推定：只有p为1，且q为0时，p➡q的真值才为0

  • 充分条件型蕴涵:

    • 如果……则；
    • 只要……就；
    • 一旦……就；
    • 有 ……就；

  • 必要条件型蕴涵：**(前件在后，后件在前)**

    • 只有……才；
    • 除非……才;
    • 除非……否则;

(4) 公式的赋值

  1. 公式的解释与真值表

  2. 命题公式的分类

(1)重言式(永真式) Tautology

  • A的每一组赋值都使A的真值为1

(2)矛盾式(永假式)

  • A的每一组赋值都使A的真值为0

(3)可满足式

  • 至少存在一组赋值使A的真值为1

(4) 命题公式的类型判断

  • 真值表最后1列全为1，则对应的命题公式为重言式
  • 真值表最后1列全为0，则对应的命题公式为矛盾式
  • 真值表最后1列有1有0，则对应的命题公式为可满足式

  • An axiom is a proposition that is assumed to be true.

(1) Axioms should be:

  • consistent
  • complete

(2) Def:

  • A set of axioms is consistent,if no proposition can be proved to be both true and false.
  • A set of axioms is said to be complete, if it can be used to prove every proposition is either true or false.
