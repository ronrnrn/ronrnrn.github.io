git 协同开发-创建分支-操作步骤

克隆（clone）项目仓库到本地

git clone <Gitee 项目仓库地址>

进入项目目录：

cd

小明创建自己的开发分支：

git checkout -b ming-dev

小明在自己的分支上进行代码修改和开发工作

以在 src 文件夹中，创建 test.txt 为例

cd src
touch test.txt

小明提交自己的更改到本地仓库：

git add .
git commit -m “(描述小明的更改)在 src 文件夹中创建了 test.txt”

小明推送（push）自己的分支到远程仓库：

git push origin ming-dev

切换到 master 主分支

git checkout master

push 到远程仓库

git push origin master
