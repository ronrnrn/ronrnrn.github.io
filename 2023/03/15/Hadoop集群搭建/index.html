JDK安装

上传到hadoop01

主机节点可用rz命令进行上传文件

在home目录下创建目录：
mkdir -p /export/servers
解压到对应目录：
tar -zxvf jdk-8u171-linux-x64.tar.gz -C /export/servers/
进入目录ls查看文件已解压安装成功：
cd /export/servers
对jdk1.8.0 171创建快捷方式为jdk：
ln -s jdk1.8.0_171 jdk
查看当前目录：
ls
查看当前路径：
pwd

配置环境变量

进入目录：
cd jdk1.8.0_171

在jdk目录下配置java环境变量:
vi /etc/profile

添加内容：
export JAVA_HOME=/export/servers/jdk
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar

让配置生效：
source /etc/profile

查看java是否安装成功：
java -version

配置免密登录

cd ~
ssh-keygen -t rsa 然后一直按回车
ssh localhost

拷贝密码到hadoop02和hadoop03：
ssh-copy-id hadoop02
ssh-copy-id hadoop03

Hadoop安装

hadoop01作为主节点：
cd /home
cd /export
进入export下创建一个目录存放hadoop:
mkdir software
然后切换目录，ls查看：
cd software/
ls
上传hadoop文件到software目录下：
rz
查看是否上传成功：
ls
解压到servers目录里：
tar -zxvf hadoop-2.10.1.tar.gz -C /export/servers
进入目录：
cd /export
cd servers/
查看是否解压成功：
ls
创建快捷方式：
ln -s hadoop-2.10.1 hadoop

配置hadoop的环境变量

vim /etc/profile

export HADOOP_HOME=/export/servers/hadoop
export PATH=$HADOOP_HOME/bin:$HADOOP_HOME/sbin:$PATH

按Esc键+:wq保存退出

使配置生效：
source /etc/profile

查看hadoop是否安装成功：
hadoop version

报错Error: JAVA_HOME is not set and could not be found.

sudo vim hadoop/etc/hadoop/hadoop-env.sh

将语句export JAVA_HOME=${JAVA_HOME}修改为：

保存后退出

再次输入：
start-dfs.sh

查看hadoop是否安装成功:
hadoop version

修改core-site和hdfs-site文件:
vim core-site.xml
vim hdfs-site.xml

初始化节点：
hdfs namenode -format

启动hadoop：
start-dfs.sh

输入jps报错：
bash: jps: 未找到命令…
解决方式：
ssh-keygen -t rsa -P ‘’ -f ~/.ssh/id_rsa
cat ~/.ssh/id_rsa.pub >> ~/.ssh/authorized_keys
chmod 0600 ~/.ssh/authorized_keys

Spark安装

下载安装

hadoop01作为主节点,/export/servers作为解压后的文件存储目录：
cd /home
cd /export
cd software/
ls查看：
ls
上传hadoop文件到software目录下：
rz
查看是否上传成功：
ls
解压到servers目录里：
tar -zxvf spark-3.3.2-bin-hadoop3.tgz -C /export/servers

配置环境变量

进入环境变量配置界面：
vi /etc/profile

新增：
export SPARK_HOME=/export/servers/spark-3.3.2-bin-hadoop3
export PATH=$SPARK_HOME/bin:$PATH

激活：
source /etc/profile

修改spark配置文件

cd spark-3.3.2-bin-hadoop3
cd conf
ls
cp spark-env.sh.template spark-env.sh

vi spark-env.sh

export JAVA_HOME=/export/servers/jdk
export SPARK_MASTER_IP=192.168.239.131
export SPARK_MASTER_PORT=7077

cd ../..返回/export/servers目录

scp -r spark-3.3.2-bin-hadoop3/ 192.168.239.131:/export/servers

cd..
./sbin/start-all.sh
jps
cd /export
cd servers

查看最终结果：
spark-shell
