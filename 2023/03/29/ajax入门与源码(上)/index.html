AJAX 概述

简介

Asynchronous JavaScript and XML
异步的 JS 和 XML

优点

  1. 无需刷新页面与服务器端进行通信
  2. 允许根据用户事件更新部分页面内容

缺点

  1. 没有浏览历史,不能回退
  2. 存在跨域问题(同源)
  3. SEO(搜索引擎优化)不友好
