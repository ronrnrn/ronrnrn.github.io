组件化开发

根据封装的思想，把页面上可重用的部分封装为组件

vue组件的构成

  • template -> 组件的模板结构(必选)
  • script -> 组件的js行为(可选)
  • style -> 组件的样式(可选)

template节点(App.vue)

vue提供的容器标签，只起到包裹性质的作用，不会被渲染成真正的DOM元素

script节点(App.vue)

可在script组件中封装组件的JavaScript业务逻辑

script-name节点(App.vue)

script-data节点(App.vue)

vue组件渲染期间需要用到的数据，可定义在data节点中

script-methods节点(App.vue)

组件中的事件处理函数，定义在methods节点中

style节点(App.vue)

在style节点中编写样式美化当前组件的UI结构

基本结构源码

style中支持less语法

步骤一：

npm install less -D

步骤二：
在style标签上添加lang=”less”属性

组件注册时名称的大小写

短横线命名法Kebab-case

例如：my-test

帕斯卡命名法或大驼峰命名法PascalCase

例如：MyTest

  • 也可以转换成短横线名称进行使用
